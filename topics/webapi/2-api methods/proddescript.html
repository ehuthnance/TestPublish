ProductDescriptions

The ProductDescriptions API can be used to manage product description information on the server.

Method

Request URL

Description

PUT

api/v7/ProductDescriptions

Adds or updates product description information from a client.

POST

api/v7/ProductDescriptions

Adds product description information from a client.

GET

api/v7/ProductDescriptions/{productCode}?description={description}

Searches products description for client and returns products description information as response.

string

•Long descriptive message

No

Defines the product culture code. Length: 1 – 10.

•fr

•es

•ja-JP
