Groups

The Groups API can be used to manage group information on the server.

Method

Request URL

Description

PUT

api/v7/Groups

Adds or updates group information from a client.

POST

api/v7/Groups

Adds group information from a client.

GET

api/v7/Groups

Receives all the group records from the database.

DELETE

api/v7/Groups/{groupCode}

Deletes the product group information for the given group code.

GET

api/v7/Groups/{groupCode}

Receives a group record from the database for the given group code.

Description

Type

string

string

No

Defines the group type code of the group. Length: 1 – 24.

Defines the date when the audit information was last modified.
