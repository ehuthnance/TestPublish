GroupDetails

The GroupDetails API can be used to manage product group information on the server.

Method

Request URL

Description

DELETE

api/v7/GroupDetails

Deletes the specified product group.

PUT

api/v7/GroupDetails

Adds or updates the information of the specified product group.

POST

api/v7/GroupDetails

Adds a product group.

GET

api/v7/GroupDetails/{groupCode}

Searches for a product group using its group code. This request retrieves information on the specified product group.

GET

api/v7/GroupDetails

Searches all group detail records from the database.

GET

api/v7/GroupDetails/{groupCode}/Products

Searches all products which belong to a specific group.

GET

api/v7/GroupDetails/Products

Searches all products which belong to a group.

The following table describes the information retrieved by the GET api/v7/GroupDetails/{groupCode} request.

Name

Description

groupCode

Specifies the group code of the product group.

groupName

Defines the name of the product group.

productCode

Specifies the code of the product group.

groupTypeCode

Specifies the type code of the product group.

groupTypeName

Specifies the type name of the product group.

suggest

Determines if the product group contains an auto-suggest flag.

order

Specifies the order of the product group.

crossReferenceKeyCode

Specifies the cross reference key code of the product group.

colorDescription

Specifies the color details of the product group.

sizeDescription

Defines the size details of the product group.

sizePattern

Defines the size pattern details of the product group.

filePath

Specifies the path of the data file containg the details of the product group.

imageName

Specifies the path of the image file used to represent the product group.

productCulture

Specifies the culture details of the product group.

shortDescription

Defines the text description of the product group.

specialComments

Defines additional information of the product group.

unitPrice

Specifies the unit price of the product group.

customerAuditDate

Specifies the date when the audit information of the product group was last modified.

culture

Specifies the culture code of the product group.
