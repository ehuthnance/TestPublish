FileOperations

The FileOperations API can be used to manage file operations information on the server.

Method

Request URL

Description

GET

api/v7/FileOperations/GetFile?fileName={fileName}

Retrieves the file, if present, in the specified path configured through sourceData with matching fileName.

HTTP request details

GET api/v7/FileOperations/GetFile?fileName={fileName}
Request information

URI parameters

The following table describes the parameter forming the HTTP request URL.

Name

Description

Type

Required

Defines the name of the file to be retrieved.

string
