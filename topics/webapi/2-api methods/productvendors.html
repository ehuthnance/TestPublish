ProductVendors

The ProductVendors API can be used to manage product vendor information on the server.

Method

Request URL

Description

PUT

api/v7/ProductVendors

Adds or updates product vendor information.

POST

api/v7/ProductVendors

Adds product vendor information.

GET

api/v7/ProductVendors/{productCode}?vendorCode={vendorCode}

Receives products vendor information for the given product code and vendor code.
