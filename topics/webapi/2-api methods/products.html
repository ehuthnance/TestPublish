Products

The Products API can be used to manage product information on the server. Product details include product tax code, price, size, color, image, brand, and description.

Method

Request URL

Description

PUT

api/v7/Products

Adds or updates product information.

POST

api/v7/Products

Accepts product information.

DELETE

api/v7/Products/{productCode}

Deletes product information.

GET

api/v7/Products/{productCode}?effectiveStartDateTime={effectiveStartDateTime}

Receives a product code and responds with product information.

GET

api/v7/Products?effectiveStartDateTime={effectiveStartDateTime}

Receives all product information as a response.

GET

api/v7/Products?productCodeList={productCodeList}

Receives a list of comma-delimited product code and responds with product information.

GET

api/v7/Products?productId={productId}&orgName={orgName}&effectiveStartDateTime={effectiveStartDateTime}

Receives the product code and orgName from client and responds with product information.

GET

api/v7/Products?searchCriteria={searchCriteria}

Searches for products based on specific criteria and responds with product information.
