SecurityV8

The newer version of the Security API that can be used to manage system login, logout, token validation, and password reset.

Method

Request URL

Description

POST

api/v8/Security/Login

Issues an authentication token to a user based on the specified orgName, user name and password.

Note: The tenant name is optional, and can be set in the custom nep-organization header.

POST

api/v8/Security/Logout

Logs out a previously issued token.

Note: The token should be set in the Authorization header.

POST

api/v8/Security/ResetPassword

Changes the current user password based on the specified user name and password within an organization context.

Note: The tenant name is optional, and can be set in the custom nep-organization header.

GET

api/v8/Security/ValidateToken

Validates an issued token.

Note: The token should be set in the Authorization header.
