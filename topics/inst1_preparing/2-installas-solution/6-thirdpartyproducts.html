Integrated third-party products

The Advanced Store POS application integrates with third-party products for the following features:

•Authorizes payments using one of the following payment solutions:

•Verifone® Flexible Integrated Payments (FiPay)

•Verifone® Point

•Cardinal® Commerce for tenders using PayPal® and PayPal Credit®

•Retailer-specific payment clients

•ACCEO Tender Retail Merchant Connect Multi (MCM)

•Accepts the following tenders:

•PayPal® and PayPal Credit®

•Europay, MasterCard®, and Visa® (EMV®) card

•Stored Value Solutions (SVS) gift card

•Accepts customer membership, loyalty, and rewards using Customer Relationship Management (CRM), such as Salesforce®.

•Accepts house card applications using retailer-specific credit application service.

•Prints or sends digital receipts using the Experian™ CheetahMail e-mail marketing software.

•Validates customer information, such as e-mail addresses and phone numbers, using the following applications:

•Experian™ QAS Pro

•IDScan.net

•Validates return transactions using the Verify® return authorization software component of the Appriss Retail™ solution

•Manages incentives the store offers to customers using Incent™ of the Appriss Retail solution

•Supports tax-free shopping using the Global Blue service provider.

Limitation

This publication is limited to the descriptions for the NCR POS application features and behavior. This publication does not describe how the POS integrates with third-party products.

Information about third-party products are available from each product’s website. For more information, consult with an NCR Representative.
