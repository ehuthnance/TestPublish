Installing the Advanced Store Solution

This section provides information about the steps to install the Advanced Store 6.8 Solution. Customers may refer to the Information Product (IP) online publications for information about the installation steps and the tools and features of Advanced Store.

Planning and defining an enterprise configuration

The Advanced Store Enterprise Solution includes the setup of several servers that support one or several stores across your enterprise. For best performance results, it is recommended to plan for the configuration that adheres with your retail requirements.

Distributed configuration

NCR Advanced Store recommends a distributed configuration when setting up your Enterprise Solution.
