Integrated NCR products

The Advanced Store Enterprise Solution integrates with the following NCR software servers:

1.Cornell–Mayo Associates™ (CMA)—Inventory Management Server (IMS)

2.Cornell–Mayo Associates™ (CMA)—Order Management Server (OMS)

3.Advanced Marketing Solution (AMS)—Logix Server

4.Advanced Marketing Solution (AMS)—Promotion/Customer Brokers

5.Web Integration Layer (SAIL)

NCR payment solution product

The Advanced Store POS application can be integrated with the Japan NCR Payment Client (NPC) to authorize payments specific to the Japanese region.

Note:  For more information on the supported cards and compatible devices, consult with an NCR Representative.

NCR Advanced Store publications

The NCR Advanced Store Installation Guides assume that the other NCR software servers exist and have already been set up and configured to integrate with the Advanced Store solution. Installation instructions for any of these servers are not described in the NCR Advanced Store Installation Guides. For more information on the installation and configuration of these servers, consult with an NCR Representative.
