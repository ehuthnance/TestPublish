Creating a Group Account in Active Directory

The Windows Active Directory Users and Computers console is used to create a security group to contain the computer accounts for the computers that are to receive a set of Group Policy settings. This task is completed on the retailer’s Active Directory server.

Note:  To complete this procedure, you must be a member of the Domain Administrators group, or otherwise be delegated permissions to create new group accounts.

Adding a new membership group

To add a new membership group in Active Directory, follow these steps:

1.Select Start→Run. The application displays the Run dialog box.

2.In the Open field, type dsa.msc, and select OK. The application displays the Active Directory Users and Computers window.

3.From the Active Directory Users and Computers window, expand the domain node.

Note:  The domain node name is dependent on the fully-qualified domain name (FQDN) set up during the Active Directory installation. In this document, the weboffice.local domain is used.

4.Select the Users node.

5.Select the Action menu.

6.Expand the New menu item.

7.Select Group. The application opens the New Object–Group window.

8.Provide a group name. For example, Managers_111.

9.Select Domain local as the group scope.

10.Select Security as the group type.

11.Select OK.

12.Repeat the steps above to create more groups, as necessary.

Adding a user to a group

To add a user to a group, follow these steps:

1.From the Active Directory Users and Computers window, expand the domain node.

Note:  The domain node name is dependent on the FQDN set during the Active Directory installation. In this document, the weboffice.local domain is used.

2.Select the Users node.

3.Right-click the group name. For example, Managers_111 group.

4.Select Properties. The application opens the properties window.

5.From the Members tab, select Add. The application opens the Select Users Contacts, Computers, Service Accounts, or Groups window.

6.From the window, enter the user login name. For example, 102.

7.Select Check Names.

8.Once the name is resolved, the user is now added to the Group Members list.

9.Repeat the steps above to add more users to the group.
